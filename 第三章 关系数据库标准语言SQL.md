# 第三章 关系数据库标准语言SQL 

## 3.1 SQL概述

- 结构化查询语言（Structured Query Language，SQL），是关系数据库的标准语言，也是一个通用的、功能极强的关系数据库语言
- 目前没有一个数据库系统能够支持 SQL 标准的所有概念和特性
- 许多软件厂商对 SQL 基本命令集还进行了不同程度的扩充和修改，又可以支持标准以外的一些功能特性

- SQL 的特点
  - 综合统一
  - 高度非过程化
  - 面向集合的操作方式
  - 以同一种语法结构提供多种使用方式
  - 语言简洁，易学易用
